    YUM (Yellowdog Updater Modified) is an open-source command-line as well as graphical-based package management tool for RPM (RedHat Package Manager) based Linux systems.

    Install a Package with YUM:

     yum -y install firefox

    Removing a Package with YUM:

     yum -y remove firefox

    Updating a Package using YUM:

     yum -y update mysql

    List a Package using YUM: Use the list function to search for the specific package with a name, it gives Installed Packages and Available Packages.

     yum list openssh

    Search for a Package using YUM: When you run yum search, the command searches for package name, summaries and description.

    yum search vim

    Get Information about a Package using YUM:

    yum info firefox

    List all Available Packages using YUM:

    yum list | less

    List all Installed Packages using YUM:

    yum list installed | less

    Yum Provides Function: Yum provides function is used to find which package a specific file belongs to.

    yum provides /etc/httpd/conf/httpd.conf

    Check for Available Updates using Yum: To find how many installed packages on your system have updates available.

    yum check-update

    exclude a particular package while updating the whole system: list of all the available packages to update using the following command.

    yum list updates | cat -n

    yum -x "package_name*" -x "package_name2*" update

    List all available Group Packages: In Linux, a number of packages are bundled into a particular group. Instead of installing individual packages with yum, you can install a particular group that will install all the related packages that belong to the group.To list all available groups.

    yum grouplist

    Install Group Packages:

    yum groupinstall 'MySQL Database'

    Update a Group Packages: To update any existing installed group packages.

    yum groupupdate 'DNS Name Server'

    Remove Group Packages:

    yum groupremove 'DNS Name Server'

    List Enabled Yum Repositories:

    yum repolist

    List all Enabled and Disabled Yum Repositories:

    yum repolist all

    Install a Package from a Specific Repository: To install a particular package from a specific enabled or disabled repository, you must use --enablerepo

    yum --enablerepo=epel install phpmyadmin

    Interactive Yum Shell: Yum utility provides a custom shell where you can execute multiple commands.

    yum shell

    Clean Yum Cache: to clean all cached files from the enabled repository.

    yum clean all

    View History of Yum:

    yum history

    Command to install/update only the security fixes:

    yum update --security

    Upgrade the system with the latest releases: upgrade your version from 7.4 to 7.8

    yum upgrade -y

    Downgrade the version: current version of "http-parser" is (2.7.1-8.el7_7.2) and revert to (2.7.1-8.el7).

    yum downgrade http-parser-2.7.1-8.el7

    RPM Commands

    RPM(Redhat Package Manager) is a command line package management utility used for installing, uninstalling, updating, querying and verifying software packages.

    GeoIP application GeoIP-1.5.0-11.el7.x86_64.rpm is a RPM Package Library for country/city/organization to IP address.

    Information of RPM package without installing: We can use -qipoption (query info package) to list the information of a package.

    rpm -qip GeoIP-1.5.0-11.el7.x86_64.rpm

    Install RPM package:

    rpm -ivh GeoIP-1.5.0-11.el7.x86_64.rpm

    Check an installed RPM package:

    rpm -q GeoIP

    List all files for particular installed RPM package:

    rpm -ql GeoIP

    List recently installed RPM packages:

    rpm -qa --last

    Install RPM package without dependencies:

    rpm -ivh --nodeps GeoIP-1.5.0-11.el7.x86_64.rpm

    Replace RPM package installed: use -ivh --replacepkgs parameters to replace a particular package installed.

    rpm -ivh --replacepkgs GeoIP-1.5.0-11.el7.x86_64.rpm

    Uninstall RPM package:

    rpm -ev GeoIP

    use -e parameters to uninstall particular package installed without dependencies don't check dependencies.

    rpm -e --nodeps GeoIP

    Upgrade RPM package installed:

    rpm -Uvh GeoIP-1.5.0-11.el7.x86_64.rpm

    Query all installed package: use -a parameters along with q to query all installed packages on the server.

    rpm -qa

    Query particular package:

    rpm -qa | grep GeoIP

    Display list of configuration file(s) for a package:

    rpm -qc GeoIP

    Query a file that belongs which RPM Package:

    rpm -qf /usr/lib64/libGeoIP.so.1.5.0

    Display list of configuration files for a command:

    rpm -qcf /usr/lib64/libGeoIP.so.1.5.0

    Get information for a particular package:

    rpm -qi GeoIP

    What dependencies a rpm file have:

    rpm -qR GeoIP

    Verify a RPM package: By using -Vp option (verify package).

    rpm -Vp GeoIP-1.5.0-11.el7.x86_64.rpm

    Verify all RPM packages:

    rpm -Va
